== Description ==
Um simples simulador de cálculo financeiro, porém eficaz.

== Installation ==
Faça o upload do plugin "Simulador de investimento" para seu blog, ative-o.

== Frequently Asked Questions ==

= Como utilizar o shortcode =
Insira a tag [SIMULADOR_DE_INVESTIMENTO] em páginas, posts e será exibido o formulário.

= Como utilizar através do widget =
Vá até widgets e adicione o widget "Simulador de Investimento" nas posições disponíveis em seu tema.

= Como utilizar o plugin =
Forneça os valores e envie o formulário para visualizar a simulação.
